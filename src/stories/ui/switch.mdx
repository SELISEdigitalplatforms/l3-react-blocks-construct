<Subtitle>Overview</Subtitle>

Switch toggles between two states, such as on/off or enabled/disabled.

## Variants

- **Default:** Standard toggle.

## Props

- **checked:** Controlled checked state.

- **defaultChecked:** Initial state for uncontrolled switch.

- **onCheckedChange:** Callback on state change.

- **disabled:** Disables interaction.

## Best Practices

- Use switches for immediate actions.

- Prefer checkboxes for form submissions.

- Ensure accessible labels for clarity.
