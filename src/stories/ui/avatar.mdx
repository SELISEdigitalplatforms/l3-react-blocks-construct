## Variants

- **Image:** Displays user-provided image.
- **Initials:** Displays user initials when no image is available.
- **Fallback/Icon:** Displays a default icon if image or initials are unavailable.

## Best Practices

- Always provide alt text for accessibility.
- Use consistent avatar sizes.
- Do not rely solely on avatar for critical information.
