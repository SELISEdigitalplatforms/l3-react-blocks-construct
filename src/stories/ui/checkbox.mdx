## Variants

- **Default:** Standard checkbox.
- **Disabled:** Non-interactive state.
- **Indeterminate:** Used to indicate partial selection.

## Best Practices

- Use checkboxes for multi-select options.
- Clearly label each checkbox.
- Avoid using checkboxes for single yes/no questions.
- Maintain visual distinction between checked, unchecked, and indeterminate states.
